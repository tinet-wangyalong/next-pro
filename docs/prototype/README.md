# TeamPilot 原型图

打开 `index.html` 即可查看基于 `docs/PROTOTYPE_SPEC.md` 生成的低保真原型图。

## 包含页面

- [ ] 登录页
- [ ] 注册页
- [ ] 仪表盘
- [ ] 组织切换
- [ ] 成员管理
- [ ] 知识库列表
- [ ] 知识条目详情
- [ ] 任务列表
- [ ] 任务详情
- [ ] AI 助手
- [ ] 组织设置
- [ ] 审计日志

## 使用方式

- [ ] 先按原型图手写静态 UI。
- [ ] 再逐个接入真实 API。
- [ ] 每个页面都保留 loading、empty、error 状态。
- [ ] 后续 Next.js 实现时，可把每个原型区域拆成独立页面和组件。
