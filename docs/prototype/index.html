核心原则

所有业务数据都需要绑定组织编号。查询列表、详情、更新和删除时， 后端都必须根据当前用户的成员关系校验组织访问权限。

前端切换组织后，应刷新当前页面数据；后端不能信任前端传入的角色信息。
