SaaS 多租户数据隔离笔记

创建人：Wangyalong · 最后更新：10 分钟前

编辑 删除

核心原则

所有业务数据都需要绑定 organizationId。查询列表、详情、更新和删除时， 后端都必须根据当前用户的 membership 校验组织访问权限。

前端切换组织后，应刷新当前页面数据；后端不能信任前端传入的角色信息。
